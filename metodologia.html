Cómo son las sesiones

Las sesiones se desarrollan de forma clara y ordenada, priorizando la comprensión práctica de cada herramienta y respetando el ritmo de aprendizaje de cada persona.

Nuestro enfoque de accesibilidad

Todas las sesiones están diseñadas bajo principios de accesibilidad, utilizando lectores de pantalla, atajos de teclado y explicaciones detalladas.

Pasos de aprendizaje

  1. Diagnóstico inicial de conocimientos y necesidades
  2. Definición de objetivos por sesión
  3. Explicación paso a paso
  4. Práctica supervisada
  5. Seguimiento y ajustes personalizados

Requisitos para empezar

  • Computadora o dispositivo móvil
  • Conexión a Internetr
  • Disponibilidad de tiempo
  • Interés por aprender tecnología accesible
